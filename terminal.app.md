## Key Mappings

| KEY                  | ACTION                            |
|----------------------|-----------------------------------|
| control cursor left  | \033[1;6D                         |
| control cursor right | \033[1;6C                         |
| end                  | \033[4~                           |
| home                 | \033[1~                           |
| page down            | \033[6~                           |
| page up              | \033[5~                           |
| shift page down      | scroll to next page in buffer     |
| shift page up        | scroll to previous page in buffer |
